[PHASE 1] Python Application — Raspberry Pi OS Lite Prototype
─────────────────────────────────────────────────────────────
  ✓ app/totp.py       — native RFC 6238 (HMAC-SHA1/256/512), no external dependency
  ✓ app/storage.py    — LUKS management via cryptsetup, RAM DB copy (tmpfs),
                        multi-pass wipe on shutdown
  ✓ app/rtc.py        — DS3231 I2C driver (read, write, system clock sync)
  ✓ app/config.py     — centralised hardware constants (GPIO, SPI, colours…)
  ✓ app/main.py       — main loop, 5-min inactivity timeout, SIGTERM/SIGINT handling

  UI layer (state machine, ST7789 240×240 via SPI + Pillow):
  ✓ ui/screen.py      — ST7789 RGB565 driver, backlight control
  ✓ ui/input.py       — GPIO polling, long SELECT press detection (2 s)
  ✓ views/unlock.py   — 8-digit PIN entry (UP/DOWN/SELECT)
  ✓ views/home.py     — scrollable account list
  ✓ views/code.py     — TOTP code display + 30 s countdown bar
  ✓ views/add_entry.py — character-by-character entry (label + Base32 secret)
  ✓ views/add_method.py — choose between manual entry and QR scan
  ✓ views/qr_scan.py  — import via Pi camera (picamera2 + pyzbar, optional)
  ✓ views/entry_menu.py — per-entry menu: Back / Delete
  ✓ views/settings.py — set time, Wipe & Lock, About, Shutdown
  ✓ views/confirm.py  — generic Yes/No confirmation dialog
  ✓ views/error.py    — error display with dismissal

  PC-side tools:
  ✓ tools/provision_data_sd.sh  — initialise SD #2 (LUKS2 + SQLite)
  ✓ tools/add_totp_entry.py     — add accounts from a Linux PC

[PHASE 2] OS Hardening — Minimal Buildroot Image
─────────────────────────────────────────────────
  ✓ buildroot/configs/airtopt_defconfig     — ARM Cortex-A53, BusyBox init,
                                              Python 3, Pillow, cryptsetup, SQLite
  ✓ buildroot/board/linux-disable-net.config — removes brcmfmac, cfg80211,
                                               BT, USB_NET at compile time
  ✓ buildroot/board/config.txt              — disable-wifi, disable-bt, SPI, I2C,
                                               gpu_mem=16, kernel=zImage
  ✓ buildroot/board/cmdline.txt             — rootfs mounted ro, loglevel=3
  ✓ buildroot/board/post_build.sh           — copies app/, verifies no swap,
                                               removes residual network files
  ✓ buildroot/board/genimage.cfg            — flashable SD image (64 MB FAT32 boot
                                               + 256 MB ext4 rootfs = ~320 MB)
  ✓ buildroot/board/post_image.sh           — dm-verity + initramfs + genimage
  ✓ rootfs_overlay/etc/fstab               — rootfs ro, /tmp /run /var as tmpfs
  ✓ rootfs_overlay/etc/inittab             — BusyBox init, app respawn
  ✓ rootfs_overlay/etc/init.d/S02tmpfs     — create tmpfs mount directories
  ✓ rootfs_overlay/etc/init.d/S05rtc       — sync clock from DS3231 at boot
  ✓ rootfs_overlay/etc/init.d/S99wipe      — wipe /tmp/totp.db at shutdown
                                              (even after SIGKILL on the app)
  ✓ buildroot/package/python-rpi-gpio/      — Buildroot recipe for RPi.GPIO

[PHASE 3] Advanced Security
────────────────────────────
  ✓ buildroot/board/linux-verity.config     — DM_VERITY=y, BLK_DEV_INITRD=y
  ✓ buildroot/board/initramfs/init          — init with __ROOT_HASH__ injected
                                              at build time; refuses to boot if
                                              the rootfs has been tampered with
  ✓ buildroot/board/post_image.sh           — veritysetup format → extract root
                                              hash → inject into initramfs →
                                              create cpio.gz → call genimage
  ✓ tools/secure_boot/generate_key.sh      — RSA-2048 key pair generation (offline)
  ✓ tools/secure_boot/sign_image.sh        — sign kernel + initramfs + firmware
  ✓ tools/secure_boot/burn_otp.sh          — OTP fuse programming (triple
                                              confirmation, irreversible)

  Full chain of trust:
    OTP → start.elf → kernel → initramfs (embedded hash) → verified rootfs
